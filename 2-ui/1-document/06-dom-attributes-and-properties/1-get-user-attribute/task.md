# Отримайте атрибут

Напишіть код, щоб вибрати елемент з атрибутом `data-widget-name` з документа та прочитати його значення.

```html run
<!DOCTYPE html>
<html>
<body>

  <div data-widget-name="menu">Виберіть жанр</div>

  <script>
    /* ваш код */
  </script>
</body>
</html>
```
