# Порахуйте властивості

Напишіть функцію `count(obj)`, що повертає кількість властивостей об’єкта:

```js
let user = {
  name: 'Іван',
  age: 30
};

alert( count(user) ); // 2
```

Намагайтеся зробити код якомога коротшим.

P.S. Ігноруйте символьні властивості, враховуйте лише “звичайні”.
