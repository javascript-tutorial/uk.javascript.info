# Перепишіть 'if..else' на '?'

Перепишіть `if..else`, використовуючи декілька тернарних операторів `'?'`.

Для зручності читання рекомендується розділити код на кілька рядків.

```js
let message;

if (login == 'Працівник') {
  message = 'Привіт';
} else if (login == 'Директор') {
  message = 'Вітаю';
} else if (login == '') {
  message = 'Немає логіну';
} else {
  message = '';
}
```
