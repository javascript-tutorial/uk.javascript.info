# Яке значення виведе цикл "for"?

Для кожного циклу запишіть, які значення він виведе. Потім порівняйте з відповіддю.

Обидва цикли виведуть одинакові числа чи ні?

1. Постфіксна форма:

    ```js
    for (let i = 0; i < 5; i++) alert( i );
    ```
2. Префіксна форма:

    ```js
    for (let i = 0; i < 5; ++i) alert( i );
    ```
