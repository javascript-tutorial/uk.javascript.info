# Яке значення виведе цикл "while"?

Запишіть для кожного циклу значення, які він виведе. Потім порівняйте з відповіддю.

Чи виводять обидва цикли однакові значення?

1. Префіксна форма `++i`:

    ```js
    let i = 0;
    while (++i < 5) alert( i );
    ```
2. Постфіксна форма `i++`

    ```js
    let i = 0;
    while (i++ < 5) alert( i );
    ```
