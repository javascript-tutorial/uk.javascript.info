Відповідь: `null`.


```js run
function f() {
  alert( this ); // null
}

let user = {
  g: f.bind(null)
};

user.g();
```

Контекст прив’язаної функції жорстко-фіксований. Немає способу змінити це в подальшому.

Так чином в той час як ми запускаємо `user.g()`, функція `f` викликається з `this=null`.
