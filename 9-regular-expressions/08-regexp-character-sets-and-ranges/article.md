# –ù–∞–±–æ—Ä–∏ —Ç–∞ –¥—ñ–∞–ø–∞–∑–æ–Ω–∏ [...]

–î–µ–∫—ñ–ª—å–∫–∏ —Å–∏–º–≤–æ–ª—ñ–≤, –∞–±–æ –∫–ª–∞—Å–∏ —Å–∏–º–≤–æ–ª—ñ–≤ –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ –∫–≤–∞–¥—Ä–∞—Ç–Ω–∏—Ö –¥—É–∂–æ–∫ `[‚Ä¶]` –æ–∑–Ω–∞—á–∞—é—Ç—å "—à—É–∫–∞—Ç–∏ –±—É–¥—å-—è–∫–∏–π —Å–∏–º–≤–æ–ª –∑-–ø–æ–º—ñ–∂ –∑–∞–¥–∞–Ω–∏—Ö".

## –ù–∞–±–æ—Ä–∏

–î–æ –ø—Ä–∏–∫–ª–∞–¥—É, `pattern:[eao]` –æ–∑–Ω–∞—á–∞—î –±—É–¥—å-—è–∫–∏–π –∑ —Ç—Ä—å–æ—Ö —Å–∏–º–≤–æ–ª—ñ–≤: `'a'`, `'e'`, or `'o'`.

–£ —Ç–∞–∫–∏–π —Å–ø–æ—Å—ñ–± –∑–∞–ø–∏—Å—É—î—Ç—å—Å—è —Ç–∞–∫ –∑–≤–∞–Ω–∏–π *–Ω–∞–±—ñ—Ä*. –ù–∞–±—ñ—Ä –º–æ–∂–µ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏—Å—å –≤ —Ä–µ–≥—É–ª—è—Ä–Ω–∏—Ö –≤–∏—Ä–∞–∑–∞—Ö —Ä–∞–∑–æ–º –∑—ñ –∑–≤–∏—á–∞–π–Ω–∏–º–∏ —Å–∏–º–≤–æ–ª–∞–º–∏:

```js run
// –∑–Ω–∞–π—Ç–∏ [t –∞–±–æ m], –∞ –ø–æ—Ç—ñ–º "op"
alert( "Mop top".match(/[tm]op/gi) ); // "Mop", "top"
```

–ó–≤–µ—Ä–Ω—ñ—Ç—å —É–≤–∞–≥—É, —â–æ –Ω–µ–∑–≤–∞–∂–∞—é—á–∏ –Ω–∞ —Ç–µ, —â–æ –≤ –Ω–∞–±–æ—Ä—ñ –≤–∫–∞–∑–∞–Ω–æ –¥–≤–∞ —Å–∏–º–≤–æ–ª–∏, –≤ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ñ —î —Å–ø—ñ–≤–ø–∞–¥—ñ–Ω–Ω—è –ª–∏—à–µ –ø–æ –æ–¥–Ω–æ–º—É –∑ –Ω–∏—Ö.

–¢–æ–∂ –Ω–∏–∂—á–µ–Ω–∞–≤–µ–¥–µ–Ω–∏–π –ø—Ä–∏–∫–ª–∞–¥ –Ω–µ –∑–Ω–∞–π–¥–µ —Å–ø—ñ–≤–ø–∞–¥—ñ–Ω—å:

```js run
// –∑–Ω–∞–π—Ç–∏ "V", –ø–æ—Ç—ñ–º [o —á–∏ i], –ø–æ—Ç—ñ–º "la"
alert( "Voila".match(/V[oi]la/) ); // null, no matches
```

–®–∞–±–ª–æ–Ω —à—É–∫–∞—î:

- `pattern:V`,
- –ø–æ—Ç—ñ–º *–æ–¥–Ω–∞* –∑ –ª—ñ—Ç–µ—Ä –Ω–∞–±–æ—Ä—É `pattern:[oi]`,
- –ø–æ—Ç—ñ–º `pattern:la`.

–†–µ–∑—É–ª—å—Ç–∞—Ç–æ–º —Ç–∞–∫–æ–≥–æ –ø–æ—à—É–∫—É –º–æ–≥–ª–∏ –± —Å—Ç–∞—Ç–∏ –≤–∞—Ä—ñ–∞–Ω—Ç–∏ `match:Vola` –∞–±–æ `match:Vila`.

## –î—ñ–∞–ø–∞–∑–æ–Ω–∏

–ö–≤–∞–¥—Ä–∞—Ç–Ω—ñ –¥—É–∂–∫–∏ —Ç–∞–∫–æ–∂ –º–æ–∂—É—Ç—å –º—ñ—Å—Ç–∏—Ç–∏ *–¥—ñ–∞–ø–∞–∑–æ–Ω–∏ —Å–∏–º–≤–æ–ª—ñ–≤*.

–î–æ –ø—Ä–∏–∫–ª–∞–¥—É, `pattern:[a-z]` —à—É–∫–∞—Ç–∏–º–µ —Å–∏–º–≤–æ–ª—É –≤ –¥—ñ–∞–ø–∞–∑–æ–Ω—ñ –≤—ñ–¥ `a` –¥–æ `z`, –∞ `pattern:[0-5]` –¥–æ—Ä—ñ–≤–Ω—é—î —Ü–∏—Ñ—Ä–∞–º –≤ –¥—ñ–∞–ø–∞–∑–æ–Ω—ñ –≤—ñ–¥ `0` –¥–æ `5`.

–í –Ω–∏–∂—á–µ–Ω–∞–≤–µ–¥–µ–Ω–æ–º—É –ø—Ä–∏–∫–ª–∞–¥—ñ –º–∏ —à—É–∫–∞—Ç–∏–º–µ–º–æ –ª—ñ—Ç–µ—Ä—É `"x"` –∑–∞ —è–∫–æ—é —Å–ª—ñ–¥—É—é—Ç—å –¥–≤—ñ —Ü–∏—Ñ—Ä–∏, –∞–±–æ –ª—ñ—Ç–µ—Ä–∏ –≤—ñ–¥ `A` –¥–æ `F`:

```js run
alert( "Exception 0xAF".match(/x[0-9A-F][0-9A-F]/g) ); // xAF
```

–¢—É—Ç `pattern:[0-9A-F]` –º–∞—î –≤ —Å–æ–±—ñ –æ–¥—Ä–∞–∑—É –¥–≤–∞ –¥—ñ–∞–ø–∞–∑–æ–Ω–∏: –≤—ñ–Ω —à—É–∫–∞—î —Å–∏–º–≤–æ–ª, —è–∫–∏–π —î –∞–±–æ —Ü–∏—Ñ—Ä–æ—é –≤—ñ–¥ `0` –¥–æ `9` , –∞–±–æ –ª—ñ—Ç–µ—Ä–æ—é –≤—ñ–¥ `A` –¥–æ `F`.

–Ø–∫–±–∏ –º–∏ –∑–∞—Ö–æ—Ç—ñ–ª–∏ —à—É–∫–∞—Ç–∏ –ª—ñ—Ç–µ—Ä–∏ –Ω–µ —Ç—ñ–ª—å–∫–∏ –≤–µ—Ä—Ö–Ω—å–æ–≥–æ, –∞ –π –Ω–∏–∂–Ω—å–æ–≥–æ —Ä–µ–≥—ñ—Å—Ç—Ä—É, –º–∏ –º–æ–≥–ª–∏ –± –¥–æ–¥–∞—Ç–∏ –¥—ñ–∞–ø–∞–∑–æ–Ω `a-f`: `pattern:[0-9A-Fa-f]`. –ê–±–æ –¥–æ–¥–∞—Ç–∏ —Ñ–ª–∞–≥ `pattern:i`.

–ö—Ä—ñ–º —Ç–æ–≥–æ, –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ `[‚Ä¶]` –º–∏ –º–æ–∂–µ–º–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ —Å–∏–º–≤–æ–ª—å–Ω—ñ –∫–ª–∞—Å–∏.

–î–æ –ø—Ä–∏–∫–ª–∞–¥—É, —è–∫—â–æ –º–∏ –∑–∞—Ö–æ—á–µ–º–æ –∑–Ω–∞–π—Ç–∏ —Å–∏–º–≤–æ–ª "—Å–ª–æ–≤–∞"  `pattern:\w` , –∞–±–æ –¥–µ—Ñ—ñ—Å `pattern:-`, –Ω–∞–±—ñ—Ä –≤–∏–≥–ª—è–¥–∞—Ç–∏–º–µ –Ω–∞—Å—Ç—É–ø–Ω–∏–º —á–∏–Ω–æ–º `pattern:[\w-]`.

–ö–æ–º–±—ñ–Ω—É–≤–∞—Ç–∏ –¥–µ–∫—ñ–ª—å–∫–∞ –∫–ª–∞—Å—ñ–≤ —Ç–µ–∂ –º–æ–∂–ª–∏–≤–æ, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥ `pattern:[\s\d]` –æ–∑–Ω–∞—á–∞—î "–ø—Ä–æ–±—ñ–ª, –∞–±–æ —Ü–∏—Ñ—Ä–∞".

```smart header="–°–∏–º–≤–æ–ª—å–Ω—ñ –∫–ª–∞—Å–∏ —Ü–µ –ª–∏—à–µ —Å–∫–æ—Ä–æ—á–µ–Ω–Ω—è –¥–ª—è –¥–µ—è–∫–∏—Ö –Ω–∞–±–æ—Ä—ñ–≤ —Å–∏–º–≤–æ–ª—ñ–≤"
–î–æ –ø—Ä–∏–∫–ª–∞–¥—É:

- **\d** -- —Ü–µ —Ç–µ —Å–∞–º–µ, —â–æ –π `pattern:[0-9]`,
- **\w** -- —Ü–µ —Ç–µ —Å–∞–º–µ, —â–æ –π `pattern:[a-zA-Z0-9_]`,
- **\s** -- —Ü–µ —Ç–µ —Å–∞–º–µ, —â–æ –π `pattern:[\t\n\v\f\r ]`, –ø–ª—é—Å –¥–µ–∫—ñ–ª—å–∫–∞ —ñ–Ω—à–∏—Ö —Ä—ñ–¥–∫—ñ—Å–Ω–∏—Ö –ø—Ä–æ–±—ñ–ª—å–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤ Unicode.
```

### –ü—Ä–∏–∫–ª–∞–¥: \w –≤ —ñ–Ω—à–∏—Ö –º–æ–≤–∞—Ö —Å–≤—ñ—Ç—É

–û—Å–∫—ñ–ª—å–∫–∏ —Å–∏–º–≤–æ–ª—å–Ω–∏–π –∫–ª–∞—Å `pattern:\w` —Ü–µ –ª–∏—à–µ —Å–∫–æ—Ä–æ—á–µ–Ω–∏–π –∑–∞–ø–∏—Å `pattern:[a-zA-Z0-9_]`, –≤—ñ–Ω –Ω–µ –∑–º–æ–∂–µ –∑–Ω–∞–π—Ç–∏ –∫–∏—Ç–∞–π—Å—å–∫—ñ —ñ—î—Ä–æ–≥–ª—ñ—Ñ–∏, –ª—ñ—Ç–µ—Ä–∏ –∫–∏—Ä–∏–ª–∏—Ü–µ—é —Ç–æ—â–æ.

–Ü—Å–Ω—É—î —Å–ø–æ—Å—ñ–± –Ω–∞–ø–∏—Å–∞—Ç–∏ –±—ñ–ª—å—à —É–Ω—ñ–≤–µ—Ä—Å–∞–ª—å–Ω–∏–π —à–∞–±–ª–æ–Ω, —â–æ –≤–∫–ª—é—á–∞—Ç–∏–º–µ –≤ —Å–µ–±–µ –±—É–∫–≤–µ–Ω–Ω—ñ —Å–∏–º–≤–æ–ª–∏ –±—É–¥—å-—è–∫–æ—ó –º–æ–≤–∏ —Å–≤—ñ—Ç—É. –¶–µ –ª–µ–≥–∫–æ —Ä–µ–∞–ª—ñ–∑—É–≤–∞—Ç–∏ –∑–∞–≤–¥—è–∫–∏ –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—è–º Unicode: `pattern:[\p{Alpha}\p{M}\p{Nd}\p{Pc}\p{Join_C}]`.

–î–∞–≤–∞–π—Ç–µ —Ä–æ–∑—à–∏—Ñ—Ä—É—î–º–æ —Ü–µ–π —à–∞–±–ª–æ–Ω. –ü–æ–¥—ñ–±–Ω–æ –¥–æ `pattern:\w`, –º–∏ —Å—Ç–≤–æ—Ä—é—î–º–æ —Å–≤—ñ–π –¥—ñ–∞–ø–∞–∑–æ–Ω, —è–∫–∏–π –≤–∫–ª—é—á–∞—î –≤ —Å–µ–±–µ —Å–∏–º–≤–æ–ª–∏ –∑ –Ω–∞—Å—Ç—É–ø–Ω–∏–º–∏ –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—è–º–∏ Unicode:

- `Alphabetic` (`Alpha`) - –¥–ª—è –ª—ñ—Ç–µ—Ä,
- `Mark` (`M`) - –¥–ª—è –∞–∫—Ü–µ–Ω—Ç—ñ–≤,
- `Decimal_Number` (`Nd`) - –¥–ª—è —Ü–∏—Ñ—Ä,
- `Connector_Punctuation` (`Pc`) - –¥–ª—è –Ω–∏–∂–Ω—å–æ–≥–æ –ø—ñ–¥–∫—Ä–µ—Å–ª–µ–Ω–Ω—è `'_'` —ñ —Ç–æ–º—É –ø–æ–¥—ñ–±–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤,
- `Join_Control` (`Join_C`) - –¥–≤–∞ —Å–ø–µ—Ü—ñ–∞–ª—å–Ω–∏—Ö –∫–æ–¥–∏ `200c` —Ç–∞ `200d`, —è–∫—ñ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É—é—Ç—å—Å—è —É –ª—ñ–≥–∞—Ç—É—Ä–∞—Ö, –∑–æ–∫—Ä–µ–º–∞ –≤ –∞—Ä–∞–±—Å—å–∫—ñ–π –º–æ–≤—ñ.

–®–∞–±–ª–æ–Ω –≤ –¥—ñ—ó:

```js run
let regexp = /[\p{Alpha}\p{M}\p{Nd}\p{Pc}\p{Join_C}]/gu;

let str = `Hi ‰Ω†Â•Ω 12`;

// –∑–Ω–∞–π—Ç–∏ –≤—Å—ñ –ª—ñ—Ç–µ—Ä–∏ —Ç–∞ —Ü–∏—Ñ—Ä–∏:
alert( str.match(regexp) ); // H,i,‰Ω†,Â•Ω,1,2
```

–ó–≤–∏—á–∞–π–Ω–æ, –º–∏ –º–æ–∂–µ–º–æ —Ä–µ–¥–∞–≥—É–≤–∞—Ç–∏ –≤–∏—â–µ–Ω–∞–≤–µ–¥–µ–Ω–∏–π —à–∞–±–ª–æ–Ω: –¥–æ–¥–∞–≤–∞—Ç–∏ Unicode –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ, –∞–±–æ –≤–∏–¥–∞–ª—è—Ç–∏ —ó—Ö. –î—ñ–∑–Ω–∞—Ç–∏—Å—å –±—ñ–ª—å—à–µ –ø—Ä–æ Unicode –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ –º–æ–∂–Ω–∞ –∑–∞ –ø–æ—Å–∏–ª–∞–Ω–Ω—è–º <info:regexp-unicode>.

```warn header="Internet Explorer –Ω–µ –ø—ñ–¥—Ç—Ä–∏–º—É—î Unicode –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ"
Unicode –≤–ª–∞—Å—Ç–∏–≤–æ—Å—Ç—ñ `pattern:p{‚Ä¶}` –Ω–µ–¥–æ—Å—Ç—É–ø–Ω—ñ —É Internet Explorer. –í—Ç—ñ–º, —è–∫—â–æ –≤–æ–Ω–∏ –Ω–∞–º –≤—Å–µ –∂ –ø–æ—Ç—Ä—ñ–±–Ω—ñ, –º–∏ –º–æ–∂–µ–º–æ —Å–∫–æ—Ä–∏—Å—Ç–∞—Ç–∏—Å—å –±—ñ–±–ª—ñ–æ—Ç–µ–∫–æ—é [XRegExp](http://xregexp.com/).

–∞–±–æ –ø—Ä–æ—Å—Ç–æ –≤–∫–∞–∑–∞—Ç–∏ –¥—ñ–∞–ø–∞–∑–æ–Ω –ø–æ—Ç—Ä—ñ–±–Ω–∏—Ö –Ω–∞–º —Å–∏–º–≤–æ–ª—ñ–≤ –ø–µ–≤–Ω–æ—é –º–æ–≤–æ—é, –Ω–∞–ø—Ä–∏–∫–ª–∞–¥ `pattern:[–∞-—è]` –¥–ª—è –ª—ñ—Ç–µ—Ä –∫–∏—Ä–∏–ª–∏—Ü–µ—é.
```

## –î—ñ–∞–ø–∞–∑–æ–Ω–∏ –≤–∏–∫–ª—é—á–µ–Ω—å

–û–∫—Ä—ñ–º –∑–≤–∏—á–∞–π–Ω–∏—Ö –¥—ñ–∞–ø–∞–∑–æ–Ω—ñ–≤, —ñ—Å–Ω—É—é—Ç—å –¥—ñ–∞–ø–∞–∑–æ–Ω–∏ "–≤–∏–∫–ª—é—á–µ–Ω—å" —è–∫—ñ –≤–∏–≥–ª—è–¥–∞—é—Ç—å –Ω–∞—Å—Ç—É–ø–Ω–∏–º —á–∏–Ω–æ–º: `pattern:[^‚Ä¶]`.

–í–æ–Ω–∏ –≤—ñ–¥—Ä—ñ–∑–Ω—è—é—Ç—å—Å—è —Å–∏–º–≤–æ–ª–æ–º –∫–∞—Ä–µ—Ç–∫–∏ `^` –Ω–∞ –ø–æ—á–∞—Ç–∫—É —ñ –∑–Ω–∞—Ö–æ–¥—è—Ç—å –±—É–¥—å-—è–∫–∏–π —Å–∏–º–≤–æ–ª *–æ–∫—Ä—ñ–º –≤–∫–∞–∑–∞–Ω–∏—Ö –≤ –¥—ñ–∞–ø–∞–∑–æ–Ω—ñ*.

–î–æ –ø—Ä–∏–∫–ª–∞–¥—É:

- `pattern:[^aeyo]` -- –±—É–¥—å-—è–∫–∏–π —Å–∏–º–≤–æ–ª –æ–∫—Ä—ñ–º  `'a'`, `'e'`, `'y'` or `'o'`.
- `pattern:[^0-9]` -- –±—É–¥—å-—è–∫–∏–π —Å–∏–º–≤–æ–ª –æ–∫—Ä—ñ–º —Ü–∏—Ñ—Ä, —Ç–∞–∫ —Å–∞–º–æ —è–∫ —ñ `pattern:\D`.
- `pattern:[^\s]` -- –±—É–¥—å-—è–∫–∏–π –Ω–µ –ø—Ä–æ–±—ñ–ª—å–Ω–∏–π —Å–∏–º–≤–æ–ª `\S`.

–ù–∏–∂—á–µ–Ω–∞–≤–µ–¥–µ–Ω–∏–π –ø—Ä–∏–∫–ª–∞–¥ —à—É–∫–∞—î –±—É–¥—å-—è–∫–∏–π —Å–∏–º–≤–æ–ª –æ–∫—Ä—ñ–º –ª—ñ—Ç–µ—Ä –ª–∞—Ç–∏–Ω–∏—Ü–µ—é, —Ü–∏—Ñ—Ä —Ç–∞ –ø—Ä–æ–±—ñ–ª—å–Ω–∏—Ö —Å–∏–º–≤–æ–ª—ñ–≤:

```js run
alert( "alice15@gmail.com".match(/[^\d\sA-Z]/gi) ); // @ —Ç–∞ .
```

## –ï–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –≤—Å–µ—Ä–µ–¥–∏–Ω—ñ [‚Ä¶]

–ó–∞–∑–≤–∏—á–∞–π, –∫–æ–ª–∏ –º–∏ —Ö–æ—á–µ–º–æ –∑–Ω–∞–π—Ç–∏ –æ–¥–∏–Ω –∑—ñ —Å–ø–µ—Ü—Å–∏–º–≤–æ–ª—ñ–≤, –Ω–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–æ –µ–∫—Ä–∞–Ω—É–≤–∞—Ç–∏ –π–æ–≥–æ –Ω–∞—Å—Ç—É–ø–Ω–∏–º —á–∏–Ω–æ–º `pattern:\.`. –¢–æ–±—Ç–æ, —è–∫—â–æ –Ω–∞–º –ø–æ—Ç—Ä—ñ–±–Ω–∞ –∑–≤–æ—Ä–æ—Ç–Ω—è –∫–æ—Å–∞ —Ä–∏—Å–∫–∞, –º–∏ –º–∞—î–º–æ –ø–∏—Å–∞—Ç–∏ `pattern:\\`, —ñ  —Ç–∞–∫ –¥–∞–ª—ñ.

–í –∫–≤–∞–¥—Ä–∞—Ç–Ω–∏—Ö –¥—É–∂–∫–∞—Ö –º–∏ –º–æ–∂–µ–º–æ –≤–∏–∫–æ—Ä–∏—Å—Ç–æ–≤—É–≤–∞—Ç–∏ –≤–µ–ª–∏–∫—É –∫—ñ–ª—å–∫—ñ—Å—Ç—å —Å–ø–µ—Ü—Å–∏–º–≤–æ–ª—ñ–≤ –±–µ–∑ –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è:

- –°–∏–º–≤–æ–ª `pattern:. + ( )` –Ω–µ –ø–æ—Ç—Ä–µ–±—É—î –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è.
- –î–µ—Ñ—ñ—Å `pattern:-` –Ω–µ –ø–æ—Ç—Ä–µ–±—É—î –µ–∫—Ä–∞–Ω—É–≤–∞–Ω–Ω—è –Ω–∞ –ø–æ—á–∞—Ç–∫—É, –∞–±–æ –≤ –∫—ñ–Ω—Ü—ñ (—Ç–æ–±—Ç–æ –∫–æ–ª–∏ –Ω–µ –º–æ–∂–µ –æ–∑–Ω–∞—á–∞—Ç–∏ –¥—ñ–∞–ø–∞–∑–æ–Ω).
